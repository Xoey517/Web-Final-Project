关于星城

一座来了就不想走的城市

长沙，湖南省会，别称“星城”。这里有三千年不变的城址，也有凌晨三点依然人声鼎沸的解放西路。

从岳麓山的千年文脉，到街头巷尾的烟火气，长沙用它的热辣与包容，等待着每一位旅人的到来。


必玩景点

橘子洲头

独立寒秋，湘江北去。在青年毛泽东雕像前，感受伟人情怀与湘江壮阔。

岳麓书院

惟楚有材，于斯为盛。中国四大书院之一，探寻千年湖湘文化的源头。

坡子街美食

臭豆腐、糖油粑粑、口味虾。在这里，唯有美食与爱不可辜负。
